## Features:
- Date & Time 
- Timer
- Chronometer
- Alarm, difined by:
    - start time
    - number of repetitions
    - interval of repetition in minutes

## Observations:
- Built with the Arduino framework
- No pin actions are implemented other than the ones for the LCD
